# 智慧消防驾驶舱 · 应急指挥模式 PRD

## 1. 页面概述

| 属性 | 值 |
|---|---|
| 页面名称 | 智慧消防大屏 · 应急指挥模式（一期） |
| 文件路径 | `一期页面/emergency.html` |
| 页面截图 | `../1.0页面截图/emergency.png` |
| 模式说明 | 应急指挥模式——以橙色为主色调，聚焦火情响应、调度指挥、实时监控。页面加载后自动切换至应急指挥模式。 |

## 3. 交互方式

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 3D旋转/缩放/平移 | 鼠标操作 | 同巡航模式 |
| 双击楼层 | 双击楼层墙体 | 进入BIM楼层详情 |
| 悬浮/锁定信息卡 | 鼠标操作 | 同巡航模式 |
| 确认告警 | 点击告警横幅按钮 | Toast"已确认告警" |
| 接管指挥 | 点击告警横幅按钮 | Toast"已接管指挥权" |
| 呼叫联系人 | 点击电话按钮 | Toast"正在呼叫xxx..." |
| 一键调度 | 点击按钮 | 打开应急调度弹窗 |
| 切换推演时间 | 点击烟气时间按钮 | 更新3D烟气模型 |
| 摄像头卡片 | 点击 | Toast"正在调取CAM-xxx画面" |
| 摄像头轮播 | 鼠标悬浮 | 暂停自动滚动 |
