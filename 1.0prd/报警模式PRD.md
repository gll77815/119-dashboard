# 智慧消防驾驶舱 · 报警模式 PRD

## 1. 页面概述

| 属性 | 值 |
|---|---|
| 页面名称 | 智慧消防驾驶舱 · 报警模式（一期） |
| 文件路径 | `一期页面/alarm.html` |
| 页面截图 | `../1.0页面截图/alarm.png` |
| 模式说明 | 报警响应模式——聚焦火情告警详情、区域设备状态、应急通讯联动。页面加载后自动切换至报警模式，自动打开B1楼层详情。 |

| 主色调 | 红色 #E24B4A |

## 3. 交互方式

### 3.1 3D主视图交互

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 旋转视角 | 鼠标左键拖拽 | 球坐标系环绕旋转 |
| 平移视角 | 鼠标右键拖拽 | 目标点平移 |

| 锁定信息卡 | 单击3D对象 | 卡片锁定 |

## 5. 报警模式自动行为

页面加载后自动执行以下操作：
1. 自动切换至报警模式（`setMode('alarm')`）
2. 自动开启图层：视频监控(L3)、人员热力(L4)、逃生路径(L5)、防排烟(LE)、防火门(LF)
3. 自动打开B1楼层详情视图
4. 底栏告警数显示为3（红色高亮）
5. 告警横幅自动显示
