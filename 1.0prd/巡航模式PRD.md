# 智慧消防驾驶舱 · 巡航模式 PRD

## 1. 页面概述

| 属性 | 值 |
|---|---|
| 页面名称 | 智慧消防驾驶舱 · 巡航模式（一期） |
| 文件路径 | `一期页面/cruise.html` |
| 页面截图 | `../1.0页面截图/cruise.png` |
| 模式说明 | 巡航模式——全楼层设备状态巡视，标注异常点位。面向消控室值班员的日常监控入口。 |
| 技术栈 | Three.js r128 + 原生 HTML/CSS/JS |
| 主色调 | 蓝紫色 #AF52DE |

### 布局结构

```
┌─────────────────── TOPBAR (48px) ──────────────────┐
│ Logo | 模式Tab(巡航/报警/应急/办公) | 时间 状态      │
├──────┬────────────────────────┬──────┬──────────────┤
│      │                        │ 报警 │ 浮动工具栏    │
│ 左侧 │    Three.js 3D建筑     │ 迷你 │  场景        │
│ 信息 │       模型主视图        │ 卡片 │  图层        │
│ 面板 │                        │      │  视图        │
│(300px)│                       │(150px)│  楼层        │
│      │                        │      │  复原        │
├──────┴────────────────────────┴──────┴──────────────┤
│                   BOTTOMBAR (36px)                   │
└─────────────────────────────────────────────────────┘
```

## 3. 交互方式

### 3.1 3D主视图交互

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 旋转视角 | 鼠标左键拖拽 | 球坐标系环绕建筑旋转 |
| 平移视角 | 鼠标右键拖拽 | 目标点平移 |
| 缩放 | 鼠标滚轮 | sph.r 变化(10~90) |
| 进入楼层详情 | 双击楼层墙体 | 打开BIM楼层详情视图 |
| 复位视角 | 双击空白区域 | 复原到初始视角 |
| 悬浮信息卡 | 鼠标移入3D对象(120ms延迟) | 显示对应类型信息卡片 |
| 锁定信息卡 | 单击3D对象 | 卡片锁定，显示关闭按钮 |
| 自动旋转 | 无操作时 | 极缓慢自动旋转 |

### 3.2 信息面板交互

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 面板折叠/展开 | 点击折叠按钮 | 面板滑出/滑入 |
| 事件处置率 | 点击该行 | 打开事件列表弹窗 |
| 巡检完成率 | 点击该行 | 打开巡检计划列表弹窗 |
| 在线率 | 点击该行 | 打开监测设备列表弹窗 |
| 派单 | 点击值班人员"派单"按钮 | 打开转工单弹窗 |
| 调度 | 点击值班人员"调度"按钮 | 打开应急调度弹窗 |
| 呼叫 | 点击值班人员"呼叫"按钮 | Toast提示"正在呼叫..." |
| 巡检任务行 | 点击任务行 | 打开巡检任务详情弹窗 |
| 报警卡片 | 点击任意卡片 | 跳转到 alarm.html |

### 3.3 浮动面板交互

| 操作 | 触发方式 | 效果 |
|------|----------|------|
| 场景/图层/视图/楼层面板 | 点击工具栏按钮 | 展开/收起对应面板 |
