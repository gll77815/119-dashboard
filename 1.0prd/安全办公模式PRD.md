# 智慧消防驾驶舱 · 安全办公模式 PRD

## 1. 页面概述

| 属性 | 值 |
|---|---|
| 页面名称 | 智慧消防大屏 · 安全办公模式（一期） |
| 文件路径 | `一期页面/office.html` |
| 页面截图 | `../1.0页面截图/office.png` |
| 模式说明 | 安全办公模式——以紫色为主色调，聚焦日常巡检管理、告警统计分析、设备误报分析、安全日历。 |
| 技术栈 | Three.js r128 + 原生 HTML/CSS/JS + Canvas |
| 主色调 | 紫色 #AF52DE |

### 布局结构

```
┌─────────────────── TOPBAR (48px) ──────────────────┐
│ Logo | 模式Tab(巡航/报警/应急/办公) | 时间 状态      │
├──────┬────────────────────────┬──────┬──────────────┤
│      │  ┌──────────┐ ┌─────┐ │      │ 浮动工具栏    │
│ 左侧 │  │安全健康度│ │AI   │ │ 右侧 │  场景        │
│ 信息 │  │ 仪表盘   │ │按钮 │ │ 信息 │  图层        │
│ 面板 │  └──────────┘ └─────┘ │ 面板 │  视图        │
│(300px)│    Three.js 3D建筑    │(280px)│  楼层       │
│      │       模型主视图       │      │  复原        │
├──────┴────────────────────────┴──────┴──────────────┤
│           活动日历轮播 (100px)                        │
└─────────────────────────────────────────────────────┘
```
